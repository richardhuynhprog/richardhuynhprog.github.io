JSON Schedule Parser

A web application that dynamically loads and combines data from multiple JSON files to display a student schedule.

Key Concepts:

  • Fetches data from multiple JSON files asynchronously
  • Matches related data using IDs
  • Dynamically updates the DOM to display results

Technologies: HTML, CSS, JavaScript

This project allows users to input name and scores. High scores can be viewed on another page just click the button.
